[[cypher-comments]]
Comments
========

To add comments to your queries, use double slash. Examples:

[source,cypher]
----
MATCH (n) RETURN n //This is an end of line comment
----

[source,cypher]
----
MATCH (n)
//This is a whole line comment
RETURN n
----

[source,cypher]
----
MATCH (n) WHERE n.property = '//This is NOT a comment' RETURN n
----
