Neo4j

What is this thing? Well, Neo4j is a database...

Dashboard - overview of the database

The Dashboard provides at-a-glance monitoring of database content and activity. The numbers are estimates rather than actual values, intended for general status rather than exact reporting.

  • convenient summary of Key Performance Indicators
  • estimate of Node, Relationship and Property counts
  • more fields available in Advanced and Enterprise

That's a chart of the graph, not a graph graph.

Data browser - inspect and visualize

The default "inspection" view displays data about individual Nodes, Relationships, or the result of a Cypher query. To visualize the graph, use the upper-right icon to toggle the display, then click on greyed Nodes to expand more of the graph.

  • inspect Node, Relationship and Property values
  • "browse" the graph progresssively
  • stylize the visualization with colors, text and icons

Showing can be easier than telling when it comes to graph data.

Console - command shell

For full command of Neo4j, use the Console as an interactive shell to run Cypher queries and test REST calls.

  • run ad-hoc queries with the Cypher panel
  • explore the REST API with the HTTP panel

REPL, shell, CLI — whatever you call it, typing commands is 1337.

Neo4j Web

Through the web you have full access to Neo4j. You can run ad-hoc Cypher queries, visualize the graph, and monitor activity. There is much detail below that surface.
