Neo4j Tour

What is this thing? Well, Neo4j is a database...

Dashboard - overview of the database

The Dashboard provides at-a-glance monitoring of database content and activity.

  • convenient summary of Key Performance Indicators
  • estimate of Node, Relationship and Property counts
  • more fields available in Advanced and Enterprise

Use the Dashboard for a quick idea about what's going on with Neo4j.

Data browser - inspect and visualize

Switch between the default "inspection" view that displays tabular data. Or, enable the built-in visualization to explore the patterns of data within your graph.

  • "browse" parts of the graph
  • inspect Node, Relationship and Property values
  • visualize a part of the graph

The exploratory visualization is a great way to explain your data to friends.

Console - command (line) central

For full command of Neo4j, use the Console as an interactive shell to run Cypher queries and directly access advanced commands.

  • use Cypher to query and create graph data
  • explore the REST API with the HTTP shell
  • experiment with raw "graph traversals"

REPL, shell, CLI — whatever you call it, a written language provides great power.
