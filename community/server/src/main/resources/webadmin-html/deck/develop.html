Three Levels of API

  • Discover the graph with REST
  • Work with Cypher queries to change or query data
  • Customize advanced work with plugins

Discover - the REST inferface

  • information about the database
  • incremental graph traversal
  • basic monitoring
  • not recommended for production-level interaction

Work - with Cypher queries

  • create a node for yourself... 
    create (me { name: "Andreas" });
  • then find it.. 
    start n=node(*) where n.name! = "Andreas" return n;
