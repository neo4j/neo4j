[[cypher-getting-started]]
= Introduction to Cypher

This friendly guide will introduce you to the Cypher query language, which is used by Neo4j.

The guide will help you:

* start thinking about graphs and patterns,
* apply this knowledge to simple problems,
* learn how to write Cypher statements,
* use Cypher for loading data,
* transition from SQL to Cypher.

If you want to keep a reference on your side while reading, please see the link:cypher-refcard/[Cypher Refcard].
